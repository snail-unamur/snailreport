Insights clés

Culture d’équipe > process

Collaboration (84%), responsabilisation (76%) et autonomie (70%) dominent ; qualité (65%) et innovation (30%) suivent.

Tests & automatisation

Unitaires (94%), intégration (75%), end‑to‑end (71%) ; CI (66%), CD (81%).

IA générative

71% d’adoption, surtout génération de code (45%) et analyse qualité (23%).
